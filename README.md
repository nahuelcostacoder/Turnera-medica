# Turnera-medica

Backend desarrollado con Spring para la gestión de turnos médicos.

## Funcionalidades

 Gestión de pacientes y médicos
- Asignación de consultorios por rango de fechas
- Turnos médicos
- Obras sociales y planes
- Cálculo automático de descuentos según obra social coincidente

## Tecnologías
- Java
- Spring Boot
- Spring Data JPA
- H2
- Lombok

## Arquitectura
- Controllers
- Services
- Repositories
- DTOs
- Mappers
- Custom Exceptions
